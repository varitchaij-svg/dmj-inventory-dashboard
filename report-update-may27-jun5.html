📊 Sync ยอดขาย ZORT → Sheets

🛒 Sync ใบสั่งซื้อ PO จาก ZORT

🖼️ Sync รูปสินค้าจาก ZORT
syncZortImages() ดึงรูปสินค้าจาก ZORT มาเก็บในชีต imageUrl — trigger ทุกวันจันทร์ 05:00 รูปแสดงใน Dashboard โดยอัตโนมัติ

🆕 เพิ่มสินค้าใหม่จาก ZORT อัตโนมัติ
syncNewProductsFromZort() ตรวจสอบสินค้าที่มีใน ZORT แต่ยังไม่มีใน Sheets แล้วเพิ่มให้อัตโนมัติ — ไม่ต้องเพิ่มสินค้าใหม่ด้วยมือ

📤 โอนสต็อก Sheets → ZORT (AddTransfer)

✅ แท็บ "ส่งแล้ว" ข้ามเครื่องได้
ดึงจากชีต "รายการโอนสินค้า" แทน localStorage ทำให้ sale/FS ทุกคนเห็น status เดียวกัน และยืนยันรับของจากมือถือตัวเองได้

🐛 บั๊กใหญ่: ของหายเมื่อกด "ส่งทั้งหมด"
idempotency cache ค้าง 6 ชั่วโมง ทำให้ของไม่โอนไปคลัง แต่ถูกลบออกจาก order list — แก้ด้วยการเปลี่ยน cache key + เปลี่ยนเป็น fail-open mode
แก้
🐛 รายการสั่งของหายจากหน้าจอ
ผูก state กับ content signature (sku+qty+date) แทนเลขแถว — เพราะแถวใน Sheets อาจขยับเมื่อ refresh ทำให้ผิด row
แก้
🐛 Order หายเมื่อ GAS ตอบ non-JSON
เพิ่ม guard กัน parse error — เมื่อ GAS timeout หรือตอบ 404/HTML ระบบจะไม่ลบ order ออก แต่รายงาน error แทน
แก้
🐛 auto-heal ทิ้ง "สำเร็จ" ที่เพิ่งกด
แก้ logic ที่ทำให้ order ที่กด "สำเร็จ" เพิ่งกดถูก heal กลับมาเป็น "รอ" ใน tick ถัดไป
ใหม่
❌ ปุ่ม "ไม่ได้จัด" พร้อม confirm dialog
ปุ่มอยู่มุมขวาบน card — กดแล้วถามยืนยัน "ของหมดจริงๆ ใช่ไหม?" ถ้ายืนยันจะตั้ง ZORT stock = 0 และลบออกจากรายการ (ทำแล้วย้อนกลับไม่ได้)

📥 ยืนยันรับของ — เลื่อนลงล่างอัตโนมัติ
