🔔 LINE Notifications

ประมาณการข้อความ/เดือน (Free tier 200)

ประเภทเมื่อไหร่จำนวน/เดือน
🚨 สต็อกใกล้หมดวันละ 1 ครั้ง (ถ้ามีของต่ำ)~30
📋 สรุปเช้าวันที่มีออเดอร์ค้าง/งานพิเศษ~20
🛒 ออเดอร์หิ้วใหม่ทุกครั้งที่สั่ง "หิ้ว"~60
🎁 ปิดงาน MTOทุกครั้งที่ปิดงานพิเศษ~10
รวมประมาณ~120/เดือน ✅
🔑 Script Properties ที่ต้องตั้ง LINE_ACCESS_TOKEN, LINE_USER_ID
⚠️ Error 429 = quota เต็มเดือน รอต้นเดือนถัดไป quota รีเซ็ตเอง ระหว่างนี้ LINE ไม่เด้ง

🔧 ทดสอบ LINE

  1. เปิด GAS Editor
  2. Dropdown เลือก debugLineMessage
  3. กด Run → ดู Log ว่ามี error อะไร
  4. ถ้า Log ขึ้น LINE push error 429 = quota เต็ม รอต้นเดือน

⚙️ ตั้ง Trigger (ทำครั้งเดียว)

  1. GAS Editor → Dropdown → setupDailySummaryTrigger → Run
    ส่งสรุปเช้า วันละ 1 ครั้ง 08:00
  2. ถ้าไม่มี syncZortBoth trigger → Run setupZortStockTrigger
    sync ZORT ทุก 2 ชั่วโมง

✅ Checklist เทสต์ก่อนใช้จริง

🛒 สั่งของ

  • สั่งของ 1 รายการ → ขึ้นใน Sheet "ลำดับที่สั่งสินค้า"
  • สั่งของชิ้นเดิมซ้ำ → เห็นป้าย "สั่งแล้ว N ชิ้น"
  • สั่ง "หิ้ว" → LINE แจ้งเจ้าของ รอ ก.ค. quota รีเซ็ต

🔄 โอน & รับของ

  • โอน 1 ชิ้น → คลังลดทันที + ขึ้น Toast เขียว
  • หน้าร้านเห็นแบนเนอร์ "รอรับ"
  • กดรับ → หน้าร้านเพิ่ม
  • คลัง + หน้าร้าน รวมกันยังเท่าเดิม (ไม่หาย)

📊 สต็อก & ZORT

  • นับสต็อก แก้ตัวเลข → บันทึก → ตัวเลขเปลี่ยน
  • รอ 2 ชม. → ตัวเลขไม่กลับค่าเดิม (บั๊กที่แก้แล้ว)
  • เทียบตัวเลขในเว็บกับ ZORT ว่าตรงกัน

🖼️ รูป & UI

  • เปิดทุกแท็บ → รูปสินค้าขึ้นครบทุกหน้า
  • กดรูปแล้วขยายดูได้ (lightbox)
  • ลองทุกบทบาท → เมนูถูกต้อง
  • โหมดง่าย → เหลือ 4 เมนู ปุ่มใหญ่ขึ้น

🔔 LINE ทำต้นเดือน ก.ค.

  • รัน debugLineMessage → LINE เด้ง ไม่มี error 429
  • ปล่อยระบบ → สรุปเช้ามาวันละครั้ง (ไม่ spam)
  • ปลายเดือน เช็ค LINE ว่าไม่เกิน 200 ข้อความ

📱 ผู้ใช้จริง

  • ให้พนักงาน/แรงงานต่างด้าวลองสั่ง+รับของเองโดยไม่ต้องสอน
  • เน็ตช้า/หลุด → เว็บยังเปิดได้ (PWA offline)

🚀 Deploy

Frontend (เว็บ) — อัตโนมัติ

  1. Push โค้ดเข้า branch master
  2. Cloudflare Pages auto-deploy ภายใน 1-2 นาที

GAS (Backend) — อัตโนมัติผ่าน GitHub Actions

  1. Push appsscript_complete.gs เข้า master
  2. GitHub Actions clasp push ให้อัตโนมัติ
⚠️ Trigger ต้องตั้งเองครั้งแรก GAS deploy โค้ดให้ แต่ไม่สร้าง trigger ให้ — ต้องรัน setupDailySummaryTrigger และ setupZortStockTrigger เองครั้งเดียว
ถ้า GAS URL เปลี่ยน ต้องอัปเดต _SHEET_BASE ใน config.js แล้ว push ใหม่
